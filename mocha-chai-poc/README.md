# README #

### What is this repository for? ###

This is POC repo for API testing using Mocha-Chai

### How do I get set up? ###

**1.** Assume that you have node installed.

**2.** Clone this repo

**3.** Access the folder

**4.** Copy environtment
```sh
$ cp env.sample .env
```

**5.** Install all dependencies:
```sh
$ npm install
```

**6.** Run Mocha API test:
```sh
$ npm test
```
