# README #

### What is this repository for? ###

This is a mockup api for api testing

### How do I get set up? ###

**1.** Assume that you have node installed.

**2.** Clone this repo

**3.** Access the folder

**4.** Install all dependencies
```sh
$ npm ci
or
$ npm install
```

**5.** Run mock-up api module
```sh
$ npm run start
```
